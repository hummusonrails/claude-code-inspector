Features

Everything you need to understand
your Claude Code usage.

No telemetry, no cloud, no accounts. Just a local server that reads your JSONL files and renders the data you already have.

3D Terrain Visualization
Every project becomes a peak in a Three.js terrain. Token usage mapped to height, time on the X axis, projects on the Z axis. Orbit, zoom, click any data point.
Project Deep Dive
Expand any session to see first prompt, last prompt, model, message count, and full token breakdown with sparklines.
Context Modals
Click any data point for a tabbed modal: Overview, First Prompt, Last Prompt, Token Usage donut chart, Context Timeline.
Activity Heatmap
GitHub-style contribution grid showing when you code with Claude. Spot patterns in your AI-assisted workflow.
Cache Efficiency
See your prompt caching hit rate across every project. Know exactly how much caching saves you.
Cost & Value Analysis
Select your Claude plan and see your value multiplier. Compare API-equivalent token cost against what you actually pay. Know your ROI.
